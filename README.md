# Renaissance 文艺复兴

一个承载情感的音乐容器，让老歌的旋律在现代技术中通过"延迟满足"和"氛围重塑"重新焕发生命力。

## 项目简介

本项目是一个基于 Flutter 的桌面音乐播放器应用，融合了 Lo-Fi 氛围音效、盲盒播放模式和独特的"黄金信件"社交功能。

## 核心功能

- 🎵 **Lo-Fi 氛围引擎** - 三轨混音台，营造复古聆听体验
- 🎁 **盲盒旋律模式** - 封面随播放进度逐渐清晰，创造"重逢"惊喜
- 💌 **黄金信件** - 完整听完歌曲后解锁陌生人的感悟

## 技术栈

- Flutter 3.x + Riverpod
- fluent_ui (Windows 桌面)
- just_audio (音频处理)

## 快速开始

```bash
cd renaissance
flutter pub get
flutter run -d windows
```
